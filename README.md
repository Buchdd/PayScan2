# PayScan2 - Система переводов и оплат

Система переводов и оплат между российскими и азиатскими счетами (например, Таиланд).

## Архитектура

- **Фреймворк** - базовая оболочка и система авторизации
- **Стримы** - различные разделы бизнеса (физ. лицо, юр. лицо и т.д.)
- **Тенанты** - личные кабинеты пользователей
- **Кошельки** - счета в разных валютах (RUB, THB, CNY и т.д.)

## Технологии

- **Frontend**: React + TypeScript + Vite
- **Backend**: Node.js + Express

## Требования

- **Node.js**: версия 20.19+ или 22.12+ (рекомендуется 22.12 LTS)
- **npm**: версия 9+

> ⚠️ **Важно**: Если у вас установлена Node.js 21.x, обновите её до версии 22.12+ или используйте версию 20.19+, так как Vite 7 требует эти версии.

## Запуск проекта

### Вариант 1: Запуск в отдельных терминалах

#### 1. Запуск бэкенда (сервера)

```bash
cd server
npm install
npm run dev
```

Сервер запустится на `http://localhost:4000`

#### 2. Запуск фронтенда (клиента)

Откройте новый терминал:

```bash
cd client
npm install
npm run dev
```

Фронтенд запустится на `http://localhost:5173` (или другом порту, который укажет Vite)

### Вариант 2: Запуск через PowerShell (Windows)

Откройте два окна PowerShell:

**Окно 1 - Бэкенд:**
```powershell
cd C:\Users\Acer\Desktop\PayScan2\server
npm run dev
```

**Окно 2 - Фронтенд:**
```powershell
cd C:\Users\Acer\Desktop\PayScan2\client
npm run dev
```

## Использование

1. Откройте браузер и перейдите на `http://localhost:5173`
2. Войдите в систему (любой email и пароль - это демо-версия)
3. Выберите стрим (Ритейл или B2B)
4. Используйте функции:
   - Просмотр кошельков
   - Переводы между счетами
   - QR-оплата
   - Пополнение счетов

## Примечания

- Это демо-версия без подключения к базе данных
- Все данные хранятся в памяти (mock data)
- Авторизация работает в демо-режиме (любые данные)
- При недоступности бэкенда фронтенд автоматически использует mock данные
